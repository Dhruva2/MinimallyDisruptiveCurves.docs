Installation

  2. Steps
  3. Downloading tutorial notebook


Dependencies

Julia v1.4 or greater

Steps

  1. Open a REPL and enter the package manager (by typing ] at the prompt)

  2. If you want, make a new environment (see package management documentation for more details ), with:

] activate .

  1. Add MinimallyDisruptiveCurves.jl to your current environment with

] add https://github.com/Dhruva2/MinimallyDisruptiveCurves.jl.git

  1. If you want access to MyModelMenagerie.jl, which contains a few differential equation models with which to try out MinimallyDisruptiveCurves.jl, you can do so:

] add https://github.com/Dhruva2/MyModelMenagerie.jl.git

  1. Finally preface any julia code running in your environment, with:

using MinimallyDisruptiveCurves

  1. Feel smug that you're not using Python.

(... but you are using a superset of Python, see PyCall.jl )
